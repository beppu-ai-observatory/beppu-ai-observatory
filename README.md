# 別府AI推薦観測所 公開データ

別府AI推薦観測所は、ChatGPT・Claude・Gemini・Grokに同じ質問を繰り返し、別府の宿・グルメ・観光スポットがAIからどのように推薦されるかを毎月記録するプロジェクトです。

このリポジトリは、第三者が調査条件と公開結果を確認するための**公開専用リポジトリ**です。API観測・集計・監査のプログラム、生回答、引用URL、回答別明細、内部監査記録は含みません。

## 観測方法

2026年7月の初回観測は2026年7月11日に実施しました。次回以降は原則として毎月1日午前9時（日本時間）に観測します。

- 対象地域：大分県別府市
- 固定質問：25問
- 自由質問：当月1問
- 対象AI：OpenAI、Anthropic、Google、xAI
- 実行回数：各質問・各AIにつき3回
- 2026年7月の合計：312回答
- Web検索：すべて有効

2026年7月に使用したモデルは次のとおりです。

| 提供元 | モデルID | 追加設定 |
|---|---|---|
| OpenAI | `gpt-5.6-terra` | 追加指定なし |
| Anthropic | `claude-sonnet-5` | 追加指定なし |
| Google | `gemini-3.5-flash` | 追加指定なし |
| xAI | `grok-4.5` | `reasoning_effort=low` |

## 観測証明（2026年8月から）

毎月の観測完了直後に、`observation-proof-YYYY-MM`という変更不能なGitHub Releaseを公開します。添付する`observation-proof-YYYY-MM.json`には、観測日時、質問数、AI数、実行回数、完了回答数、モデルID、非公開証拠のSHA-256指紋だけを記録します。

生回答、request ID、引用URL、内部ログ、プログラムは公開せず、非公開リポジトリへ永久保存します。後日確認が必要になった場合、非公開証拠から指紋を再計算し、変更不能ReleaseおよびX等へ投稿した同じ指紋と照合します。

この仕組みは、公開した指紋の時点から証拠が差し替えられていないことを確認するためのものです。生回答を公開しないため、第三者が全回答の内容を直接閲覧する仕組みではありません。

## スコアリング

- 回答内1位：3点
- 回答内2～3位：2点
- 回答内4位以降：1点
- Q01（宿を1つ）：5点
- Q21（モデルコース内の登場）：一律2点
- Q19（地獄めぐり比較内の登場）：一律1点
- Q09：スコア対象外
- 自由質問：スコア対象外

同じ回答内で同一対象が重複しても二重加算しません。閉店・地域外・表記揺れ・推薦対象ではない文言は、公開前の確認工程で判定します。

## 結果の読み方

この結果は、店舗や施設の味・品質・接客・安全性を評価したランキングではありません。AIが観測時点で、どの対象を、どの順位で、どれくらい繰り返し推薦したかを集計したものです。

AIの回答には誤りや古い情報が含まれる可能性があります。営業時間、定休日、価格、所在地、アレルギー情報、営業状況などは、来訪前に必ず公式情報をご確認ください。

`status=unregistered`は、観測時点で正式名称や営業状況等の確認が完了していない候補を示します。確認済み施設の一覧として利用しないでください。

## 訂正方針

公開後に誤集計や表記誤りが確認された場合、過去版を黙って差し替えず、確認日・修正内容・理由が分かる訂正版を追加します。各月の案内ページから最新の参照値を確認できます。
